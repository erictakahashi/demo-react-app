# Demo React App

This is just a demo React app with common set up.

## Technology Stack:

- **React**
- **Vite**
- **TypeScript**
- **Material UI**
- **Storybook**
- **Vitest**

## Getting Started

1. **Clone the repository:**

```bash
git clone [repo]
```

2. Install dependencies:

```bash
npm install
```

3. Start development server:

```bash
npm run dev
```

4. Running Tests:

```bash
npm run test:unit
```

5. Storybook:

```bash
npm run storybook
```
